# Glosario del Proyecto

| Término | Definición |
| :--- | :--- |
| **Base de datos** | Sistema para almacenar y organizar datos de forma electrónica. |
| **Columna** | Campo vertical en una tabla que contiene un tipo de dato específico. |
| **Conexión** | Enlace establecido entre la aplicación Java y el servidor de datos. |
| **Controlador** | Clase que maneja la lógica y comunica la Vista con el Modelo. |
| **Driver** | Software que permite que Java se comunique con un motor de base de datos. |
| **Esquema** | Estructura que define la organización de la base de datos. |
| **Fila** | Registro individual dentro de una tabla de la base de datos. |
| **Foreign key** | Clave externa que vincula una tabla con otra. |
| **Índice** | Estructura que acelera la búsqueda de datos en una tabla. |
| **INSERT** | Comando SQL utilizado para agregar nuevos registros. |
| **JavaFX** | Biblioteca de Java para crear interfaces gráficas de usuario. |
| **JDBC** | Tecnología de Java para conectar y ejecutar consultas en bases de datos. |
| **Modelo** | Parte del sistema que gestiona los datos y las reglas de negocio. |
| **Primary key** | Identificador único para cada registro en una tabla. |
| **Query** | Consulta escrita en SQL para solicitar información a la base de datos. |
| **SELECT** | Comando SQL para recuperar o leer datos de las tablas. |
| **Tabla** | Estructura donde se guardan los datos en filas y columnas. |
| **Vista** | Parte visual del sistema con la que interactúa el usuario. |
