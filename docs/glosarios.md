# Glosario del Proyecto - Estante

Este documento define los términos técnicos clave utilizados en el desarrollo del sistema.

## Términos Técnicos

* **API:** Interfaz de programación de aplicaciones.
* **CLI:** Interfaz de línea de comandos.
* **CRUD:** Operaciones básicas (Crear, Leer, Actualizar, Borrar).
* **Dispatcher:** Componente que despacha comandos a sus funciones.
* **YAML:** Formato de archivo para configuraciones.
* **Fork:** Copia de un repositorio en tu propia cuenta.
* **Commit:** Guardar un cambio en el historial.
* **Pull Request:** Propuesta de cambios al repositorio principal.
