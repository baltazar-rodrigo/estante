# Glosario del Proyecto - Estante

Este documento define los términos técnicos clave utilizados en el desarrollo del gestor de base de datos.

## Términos Técnicos

1. **Base de datos:** Sistema organizado para almacenar y gestionar datos digitalmente.
2. **Columna:** Elemento vertical en una tabla que contiene datos de un tipo específico.
3. **Conexión:** Enlace establecido entre la aplicación y el motor de la base de datos.
4. **Controlador:** Componente que gestiona la lógica y comunica el modelo con la vista.
5. **Driver:** Software que permite al programa comunicarse con la base de datos.
6. **Esquema:** Estructura lógica que define la organización de la base de datos.
7. **Fila:** Registro individual que contiene un conjunto de datos en una tabla.
8. **Foreign key:** Clave externa que vincula dos tablas entre sí.
9. **Índice:** Estructura que mejora la velocidad de búsqueda de datos en una tabla.
10. **INSERT:** Comando SQL utilizado para agregar nuevos registros.
11. **JavaFX:** Framework utilizado para crear la interfaz gráfica de usuario.
12. **JDBC:** API de Java para ejecutar sentencias SQL y conectar con bases de datos.
13. **Modelo:** Representación de la estructura de datos y la lógica de negocio.
14. **Primary key:** Identificador único para cada registro en una tabla.
15. **Query:** Consulta realizada a la base de datos para obtener información.
16. **SELECT:** Comando SQL fundamental para recuperar datos de las tablas.
17. **Tabla:** Estructura fundamental de almacenamiento compuesta por filas y columnas.
18. **Vista:** Interfaz con la que el usuario interactúa en la aplicación.
19. **YAML:** Formato de archivo sencillo utilizado para la configuración del sistema.
20. **GitHub:** Plataforma de desarrollo colaborativo para alojar proyectos usando Git.
